Here there are the main features:

  1. Secure Registration
    • Password matching check during registration
    • Secure Password hashing with pbkdf2:sha256
  2. User Authentication
    • Flask-login used to handle sessions
    • Route protection using @login_required decorator
  3. Session Handling
    • Persistent session even after closing the browser (optional).
  4. Events Handling
    • Each user can create and visualize their own events
    • When you're seeing your events, you can mark them as done
    • After your mark them as done, you can choose to remove them from the list
  5. Database interaction
    • SQLAlchemy used to manage users and events
    • one-to-many relationship between users and events
  6. UI
    • Bootstrap used for Responsive Design
    • Flask-WTF used for HTML Modules
  7. Security
    • CSRF token used against CSRF Attacks

How can I use this app

Let me guide you into a step-by-step guide

  1. Register using the Sign Up button Above
  2. Log in into the created account
  3. Go to the "Tasklist" link
  4. Add an event from now on, give it a name and a description and click Add
  5. Now you can navigate through the dates using the timeline and see your events!
  6. You can click on your username whenever you want to go back to the profile page
